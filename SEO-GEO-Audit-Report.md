## 2. SEO טכני (ציון: 35/100)

### בעיות קריטיות

#### 2.1 חסר: robots.txt
**חומרה: קריטי**
אין קובץ robots.txt. כשהאתר יעלה לאוויר, מנועי חיפוש לא ידעו מה מותר לסרוק.

**פתרון:** יש ליצור קובץ `robots.txt` בתיקיית השורש:
```
User-agent: *
Allow: /
Sitemap: https://[domain]/sitemap.xml
```

#### 2.2 חסר: sitemap.xml
**חומרה: קריטי**
אין מפת אתר. זה חיוני לאינדוקס מהיר ומלא.

**פתרון:** יש ליצור `sitemap.xml`:
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url><loc>https://[domain]/</loc><priority>1.0</priority></url>
  <url><loc>https://[domain]/about.html</loc><priority>0.8</priority></url>
  <url><loc>https://[domain]/services-owners.html</loc><priority>0.9</priority></url>
  <url><loc>https://[domain]/services-developers.html</loc><priority>0.9</priority></url>
  <url><loc>https://[domain]/tools.html</loc><priority>0.7</priority></url>
</urlset>
```

#### 2.3 חסר: Canonical Tags
**חומרה: קריטי**
אף עמוד לא מכיל תג canonical. זה בעיה במיוחד כי קיים הקובץ `lendover.html` שמכיל תוכן כפול.

**פתרון:** להוסיף בכל עמוד ב-`<head>`:
```html
<link rel="canonical" href="https://[domain]/[page].html" />
```

#### 2.4 קובץ כפול: lendover.html
**חומרה: גבוה**
הקובץ `lendover.html` הוא גרסה ישנה של האתר עם CSS inline, תוכן כפול, וגם טופס יצירת קשר. זה יגרום לבעיות duplicate content.

**פתרון:** למחוק את `lendover.html` או להפנות אותו (301 redirect) לדף הבית.

#### 2.5 חסר: Favicon
**חומרה: בינוני**
אין הגדרת favicon. זה משפיע על זיהוי האתר בתוצאות חיפוש ובטאבים.

**פתרון:**
```html
<link rel="icon" type="image/png" href="/favicon.png" />
```

#### 2.6 שמות קבצים בעבריית עם רווחים
**חומרה: בינוני**
שמות קבצים כמו `0T2A8485 בחירה.jpg` ו-`20251112_2239_לוגו ללא רקע_remix_...png` מכילים רווחים ותווים בעברית. זה יכול לגרום לבעיות URL encoding.

**פתרון:** לשנות שמות לאנגלית: `aharon-portrait.jpg`, `logo-transparent.png`

#### 2.7 חסר: Open Graph ו-Twitter Cards
**חומרה: בינוני**
אין תגי שיתוף חברתי. כשמישהו ישתף את האתר, לא יופיע תצוגה מקדימה.

**פתרון:** להוסיף בכל עמוד:
```html
<meta property="og:title" content="אהרון לנדובר | יעוץ התחדשות עירונית" />
<meta property="og:description" content="..." />
<meta property="og:image" content="https://[domain]/og-image.jpg" />
<meta property="og:type" content="website" />
<meta property="og:locale" content="he_IL" />
<meta name="twitter:card" content="summary_large_image" />
```

#### 2.8 חסר: Preload לנכסים קריטיים
**חומרה: נמוך-בינוני**
הגופן Assistant נטען מ-Google Fonts ללא preconnect. גם קובץ הווידאו בהירו עלול להאט טעינה.

**פתרון:**
```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link rel="preload" as="video" href="iStock-2149918736.mp4" />
```

#### 2.9 חסר: דף 404 מותאם
**חומרה: נמוך**
אין דף שגיאה 404. משתמשים שיגיעו לכתובת שגויה יראו את ברירת המחדל של השרת.
